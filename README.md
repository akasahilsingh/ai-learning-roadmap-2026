## ✨ Features

- **Adaptive Curriculum** — 8–18 week roadmap that adjusts based on your background, time, and goals
- **65+ Curated Resources** — Free courses, docs, YouTube videos, tools, and guides from OpenAI, Anthropic, DeepLearning.AI, Hugging Face, fast.ai, Google, and more
- **Beginner-Friendly Resources** — Non-tech users get YouTube explainers, visual courses, and interactive guides instead of technical docs
- **Difficulty Badges** — 🟢 Beginner / 🟡 Intermediate / 🔴 Advanced tags on every resource
- **Resource Filters** — Filter resources by difficulty level and format (video, course, article, tool)
- **Estimated Time** — See how long each resource takes before you start
- **Task Progress Tracking** — Check off tasks with localStorage persistence across sessions
- **Per-Task Notes** — Add personal notes to any task
- **Progress Rings & Bars** — Visual week-level and overall progress indicators
- **Day Streak Tracking** — Fixed streak logic that counts once per calendar day
- **Skill Tree Sidebar** — Visual progression through AI skill themes
- **Export Progress** — Download your full progress as JSON
- **Onboarding Wizard** — 4-step personalization flow with step indicators
- **Zero Dependencies** — Single `index.html` file, no build step, no npm

## 🚀 Getting Started

Just open `index.html` in any modern browser. No server required.

```bash
# Clone the repo
git clone https://github.com/akasahilsingh/ai-learning-roadmap-2026.git

# Open directly
open index.html        # macOS
start index.html       # Windows
xdg-open index.html   # Linux
```
